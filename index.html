Configuração inicial

Como configurar:
  1. Acesse o Google Cloud Console
  2. Crie um projeto e ative a Google Sheets API
  3. Em "Credenciais", crie um ID do cliente OAuth 2.0 (tipo: Aplicativo da web)
  4. Em "Origens JavaScript autorizadas", adicione a URL onde este arquivo está sendo servido (ex: http://localhost:8080)
  5. Cole o ID do cliente abaixo

Nota: Para usar no celular, recomendamos hospedar no GitHub Pages e usar a URL do repositório.
